# 🌍 환경 알림이

> 아두이노와 앱인벤터를 활용하여 만든 환경 정보 알림 프로젝트입니다!

---

## 📖 프로젝트 소개

아두이노와 앱인벤터를 활용해 만든 프로젝트입니다.  
미세먼지 센서, 자외선 센서, 온습도 센서를 사용하여 주변 환경 정보를 측정하고 사용자에게 알려줍니다.

LCD 화면과 스마트폰 앱에서 현재 환경 상태를 확인할 수 있으며, LED 색깔로 좋음/보통/나쁨 상태를 쉽게 알 수 있습니다.

---

## 💡 제작 동기

일상생활에서 날씨 뉴스를 보고 야외 활동을 했지만, 실제 미세먼지 수치가 다르게 느껴지는 경우가 많았습니다.  
그래서 직접 주변 환경을 측정하고 확인할 수 있는 장치를 만들고 싶어 제작하게 되었습니다.

---

## ✨ 주요 기능

- 🌫️ 미세먼지 센서 값을 LCD에 출력
- ☀️ 자외선 센서 값을 LCD에 출력
- 🌡️ 온습도 센서 값을 앱인벤터 앱에 출력
- 📱 블루투스를 이용해 스마트폰 앱과 연결
- 🔴🟡🟢 LED 색깔로 환경 상태 표시
- 🎨 앱 배경 색깔로 좋음 / 보통 / 나쁨 표시

3. 🚀 앱을 실행합니다.
4. 📊 LED, LCD, 스마트폰 화면을 확인합니다.

# 🔍 문제 해결 과정

## ❗ 문제점
Arduino UNO는 I2C 핀이 A4, A5 한 쌍만 존재합니다.  
하지만 미세먼지 센서와 LCD가 모두 I2C 통신을 사용하여 핀 충돌 문제가 발생했습니다.

## ✅ 해결 방법
브레드보드에서 A4, A5 핀을 함께 연결하고,  
각 부품의 고유 주소(Address)를 사용하여 미세먼지 센서와 LCD를 동시에 연결했습니다.

덕분에 두 장치를 충돌 없이 사용할 수 있었습니다.
